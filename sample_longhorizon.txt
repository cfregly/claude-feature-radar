  Long-horizon chain audit on Claude Haiku 4.5. The isolated value of context editing.
  Workload: a chain of 8 incident reports, each about 40,000 tokens (a
  realistic large tool payload, e.g. a big log or trace). The agent reads them one at a
  time and counts the URGENT ones.
  ISOLATION: both runs are identical, same model, same prompt, the MEMORY TOOL ON in both,
  caching ON in both. The ONLY variable is context editing, so any difference below is
  caused by context editing alone. Claude Haiku 4.5 window: 200,000 tokens. True count: 3.

  WHAT YOU GET
    Editing OFF : finished, answer 2 (WRONG, expected 3), peak context 156,558 tokens.
    Editing ON  : finished, answer 3 (correct), peak context 34,109 tokens.

    At this payload size the editing-OFF run did not reach the 200,000 window, so both
    arms finish and (because the memory tool is on in both) both answer correctly. Context
    editing's value has not appeared yet: it shows up only when the job is heavy enough to
    hit the window. Raise --doc-tokens to see the editing-OFF arm crash.

  WHY IT FAILS WITHOUT EDITING
    The editing-OFF context climbed 1,816 -> 156,558 tokens and its
    per-turn cost climbed 23x (0.00225 -> 0.05134). The
    editing-ON arm held context near 34,109 tokens at a flat per-turn cost.

  WHERE CORRECTNESS COMES FROM (the honest split)
    The MEMORY TOOL (on in both arms) is what makes the count correct: the agent tallies into
    a durable file instead of an in-context count. CONTEXT EDITING is what makes the long run
    survivable: it bounds the window so the agent never hits the wall. Two primitives, two
    jobs. This benchmark isolates context editing only.

  HONEST COST: context editing is not a cheaper bill. Clearing rewrites the cached prefix,
  so on a job short enough to finish either way the editing-ON run can cost MORE. Its value
  is that a heavy job finishes at all. The editing-ON run here cost $0.3468.

  Reproduce: `make longhorizon` on Claude Haiku 4.5, your own key. This run cost $0.5942 total
  and took 34s wall.
