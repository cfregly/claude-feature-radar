# Cited facts

Every shipped price and fact in this repo, grounded through Claude's own Citations API. Each row was produced by handing Claude the source document with `citations.enabled=true` and asking it to quote the supporting text. The API returned a guaranteed char pointer plus the verbatim `cited_text`, which this tool re-checks resolves to the source. Regenerate with `make cite`. Sources are committed, dated snapshots under `sources/` and our own committed receipts under `edges/*/sample.txt`.
